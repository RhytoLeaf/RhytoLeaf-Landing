# Zeek Chrome Extension — Installation Guide

## Prerequisites
- Google Chrome browser (version 116 or later)

## Steps

1. **Download the extension folder**
   Download or clone the `zeek-extension-chrome` folder to your computer.

2. **Open Chrome Extensions page**
   Navigate to `chrome://extensions` in your address bar, or go to **Menu > Extensions > Manage Extensions**.

3. **Enable Developer Mode**
   Toggle the **Developer mode** switch in the top-right corner of the page.

4. **Load the extension**
   Click **Load unpacked** and select the `zeek-extension-chrome` folder.

5. **Pin the extension** (optional)
   Click the puzzle piece icon in the Chrome toolbar and pin **Zeek — RSVP Speed Reader** for quick access.

## Usage

- **Popup**: Click the Zeek icon in the toolbar to paste or type text, then click **Zeek** to open the reader in the side panel.
- **Context menu**: Select text on any page, right-click, and choose **Read with Zeek**.
- **Keyboard shortcut**: Select text and press **Alt+Z** to send it to the reader.

## Troubleshooting

- If the context menu or keyboard shortcut does nothing, reload the extension from `chrome://extensions` and refresh the page you're reading from.
- If the side panel doesn't open, make sure you're not on a `chrome://` system page — extensions can't run there.
