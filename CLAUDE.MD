# RhytoLeaf Landing Page

## Project Overview
This is the landing page for **rhytoleaf.ca** - a software development company that specializes in creating innovative applications to apply STEM (Science, Technology, Engineering, and Mathematics) in daily life.

**Mission**: Cultivating Innovators, One App at a Time

## Website Structure
The homepage (`index.html`) serves as a showcase for RhytoLeaf's portfolio of web applications currently under development.

## Technology Stack
- **Frontend Framework**: Bootstrap 5.3.3
- **CSS**: Custom styles with glassmorphism effects
- **JavaScript**: Vanilla JS + Anime.js for animations
- **Icons**: Font Awesome 6.5.1
- **Fonts**: Google Fonts (Montserrat)

## Key Features
- **Theme Toggle**: Light/Dark/Auto mode switcher
- **View Modes**: Grid and list view for app showcase
- **Responsive Design**: Mobile-first approach
- **Glassmorphism UI**: Modern frosted glass effect on containers
- **Interactive Elements**: Sparkle animation on hover

## Design Elements
- **Brand Color**: RhytoLeaf Green (`#00CE7C`)
- **Logo**: `images/logo-icon.png`
- **Background Gradients**:
  - Light mode: Blue gradient (`#83a4d4` to `#b6fbff`)
  - Dark mode: Dark gradient (`#0f2027` to `#203a43` to `#2c5364`)
